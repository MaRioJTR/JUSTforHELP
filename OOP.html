Before We Start

You need to know

This course introduces JAVA to students as an example of an object oriented programming language. The course starts with a briefing on java history and the classifications of different java editions. The concepts of classes and objects are introduced followed by a description of various types of class members.     Creating Java File

    There is general way to create .java file

    Step 1 : After opening your IDE Create a new file in your project's source directory. This is typically the .src folder or its equivalent.

    Step 2 : Name the file with a .java extension. For example, MyClass.java. Ensure that the filename matches the class name you intend to use, as Java class names should match the filename.

    Step 3 : Open the newly created .java file. Add your Java code. For example

    
                public class MyClass {
                  public static void main(String[] args) {
                      System.out.println("Hello, World!");
                  }
              }

    Step 4 : Save file and RUN

    NOTE : "During the course, you might see me replace public class MyClass with public class Main. There's no difference; it's just for you to be carful while copy-paste method.".

    Syntax

    Every line of code that runs in Java must be inside a class . And the class name should always start with an uppercase first letter. In our example, we named the class Main.

    NOTE : Java is case-sensitive: "MyClass" and "myclass" has different meaning.

    The Main Method

    The main() method is required and you will see it in every Java program: 

    
                  public static void main(String[] args)

    Any code inside the main() method will be executed. Don't worry about the keywords before and after it. You will get to know them bit by bit while course.

    For now, just remember that every Java program has a class name which must match the filename, and that every program must contain the main() method

    Arithmetic Operations

    You need to know the priority of each Arithmetic operation

    • () → operation inside the brackets has first priority to be executed
    • *,/,% → has second priority
    • +,- → is the last one

    NOTE : if all operations in the same line have same priority, it will be executed from left to right

     
                    public class Main { 
                        public static void main(String[] args) {
                          System.out.println(2 + 3 * 5);
                          System.out.println((2 + 3) * 5);
                        }
                      }

    Increment and Decrement Operator

    k++ , ++k → k = k+1

    k-- , --k → k = k-1

    The Difference is when you gonna apply the Increment and Decrement.

    Pre-incremental: ++k that mean the compiler will apply incremental in execution يعني هيضيف اثناء الطبع او التنفيذ

    Post-incremental: k++ that mean the compiler will apply incremental after execution هيضيف بعد الطبع او التنفيذ

     
                    public class Main { 
                        public static void main(String[] args) {
                          int x = 2;
                          int k = 9;                         
                          System.out.println( ++x );
                          System.out.println( k++ );
                          System.out.println( k );
                        }
                      }

    Control Statements

    Normally, Statements in program are executed one by one "Sequential Execution".

    But sometimes you need to specify the execution or the priorities "Transfer of Control".

    We categorize the "Control Statement" in two groups -> 1. Selection Control Statement 2. Repetition Control Statement

    1. Selection Statement ( If statement )

    Example

    
  if (condition) {
      printing or execution
  }

  else if (condition) {
      printing or execution
  }

  else{
      printing or execution
  }

    ADD-ons:You have to know "Rational Expression"

    == → equal to " we use it to make comparision".

    • x = 50 means store 50 in variable "x".
    • x == 50 means QUESTION, you are asking your compiler if x == 50 and the answer is "TRUE" or "FALSE" , "1" or "0".

    != → not equal to.

    < ,> → less and greater than.

    <= ,>= → less or equal to and greater or equal to.

    While Loop

    A while loop in programming allows you to repeat a block of code as long as a given condition is true. It keeps running the code inside it until the condition evaluates to false

    Example

     while (continuation condition)

 {

 Statement 1
 Statement 2
 .
 .
 .
 . 
 . 
 .

 }

    The code will execute the statements until continuation condition end or be "FALSE" value

    Program to get grade average for 5 students 

    
                import java.util.Scanner; 

                public class Main { 
                
                    public static void main(String[] args) {
                      Scanner input = new Scanner(System.in);
                      int grade = 0; // default value of grade
                      float sum =0; // default value of sum
                      int i = 1;
                      while(i<=5) 
                      {
                          System.out.println("Enter grade : ");
                          grade = input.nextInt();
                          sum = sum + grade; // or sum += grade
                          i++;
                      }
                      sum = sum / 5;
                      System.out.printf("AVG is : %f",sum);
                    }
                }

    Do While Loop

    Do while loop is similar to the while loop with the only difference that it checks for the condition after executing the statements, and therefore it is an exit-controlled loop.

    We use "do while loop" when we want to make Menu, because it appears at least 1 time

    Example

    do{
  Statement 1
  Statement 2
  . 
  . 
  . 
  . 
  Increment or decrement
}while(continuation condition);
