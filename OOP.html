Before We Start

You need to know

This course introduces JAVA to students as an example of an object oriented programming language. The course starts with a briefing on java history and the classifications of different java editions. The concepts of classes and objects are introduced followed by a description of various types of class members. The course describes the concepts of abstraction and encapsulation and how to apply these concepts when creating classes. The course describes the object oriented relationships: association, aggregation, composition, inheritance and implementation, also the concept and role of polymorphism are introduced covering method overloading, method overriding and dynamic method dispatching.

Why Use Java?

  • Java works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc.)

  • It has huge community support (tens of millions of developers)
  • Java is an object oriented language which gives a clear structure to programs and allows code to be reused, lowering development costs

    Creating Java File

    There is general way to create .java file

    Step 1 : After opening your IDE Create a new file in your project's source directory. This is typically the .src folder or its equivalent.

    Step 2 : Name the file with a .java extension. For example, MyClass.java. Ensure that the filename matches the class name you intend to use, as Java class names should match the filename.

    Step 3 : Open the newly created .java file. Add your Java code. For example

    
                public class MyClass {
                  public static void main(String[] args) {
                      System.out.println("Hello, World!");
                  }
              }

    Step 4 : Save file and RUN

    NOTE : "During the course, you might see me replace public class MyClass with public class Main. There's no difference; it's just for you to be carful while copy-paste method.".

    Syntax

    Every line of code that runs in Java must be inside a class . And the class name should always start with an uppercase first letter. In our example, we named the class Main.

    NOTE : Java is case-sensitive: "MyClass" and "myclass" has different meaning.

    The Main Method

    The main() method is required and you will see it in every Java program: 

    
                  public static void main(String[] args)

    Any code inside the main() method will be executed. Don't worry about the keywords before and after it. You will get to know them bit by bit while course.

    For now, just remember that every Java program has a class name which must match the filename, and that every program must contain the main() method

    Output (Prints)

    Unlike C, Java has different type of Print

    • 1. Using System.out.print()

      Prints text to the console without a newline at the end.

      
                    System.out.print("Hello");
                    System.out.print(" World!");

      Output:

      Hello World!


    • 2. Using System.out.println()

      Prints text to the console with a newline at the end.

      
                    System.out.println("Hello");
                    System.out.println(" World!");

      Output:

      Hello

      World!


    • 3. Using System.out.printf()

      Prints text by using format specifiers.

      
                    int age = 19;
                    String name = "Mario"
                    System.out.printf("Name = %s Age = %d",name,age);

      Output:

      Name = Mario Age = 19

    Data Types

    • String - stores text, such as "Hello". String values are surrounded by double quotes
    • int - stores integers (whole numbers), without decimals, such as 123 or -123
    • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
    • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
    • boolean - stores values with two states: true or false
